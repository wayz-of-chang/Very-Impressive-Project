== Description

This is a Rails 4.0 Application that hosts a flight map, with airlines, airports, and routes displayed on a world map.

== Set Up

To get this working, 

* Install Ruby (1.9.3) and Rails 4.0 (http://rubyonrails.org/).
* Install a SQL Database.  By default, I am using MySQL (http://www.mysql.com/).
* Set up the database by loading data found from OpenFlights (http://openflights.org/data.html).
* Pull this project.
* Navigate to the project root directory, and run rails s -e production.
